mở Docker Quickstart Terminal rồi để tự chạy
chạy câu lệnh:
docker run --net=host -ti ubuntu:16.04 bash
//Ctrl+C hoặc Ctrl+D hoặc exit để thoát 
Để docker chạy được trên win 10 thì phải chạy Hyper-V
chạy lệnh trên powershell: Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V -All
